## todo-react: 使用react、flux、gulp、browserify构建的todo-list demo

### 使用gulp进行构建

#### 安装npm依赖
  `npm install`、`gulp`

#### 查看demo
执行`gulp server`，就会自动打开[http://localhost:3333](http://localhost:3333)
